# Site Memory (Local Overlay)

This directory stores deployment-local notes that may be useful during active operations.

## Purpose

- hold temporary operator notes
- capture network quirks
- track model gaps pending graph fixes
- support handoff context

## Boundary

Site memory is graph-adjacent context, not a replacement for graph truth.
If local notes conflict with model facts, update the model and retire the note.

## Suggested local files

- `local/SITE_NOTES.md`
- `local/NETWORK_NOTES.md`
- `local/MODEL_GAPS.md`
- `local/OPERATOR_HANDOFF.md`
